# Machine Learning

This section demonstrates practical ML workflows including supervised learning, unsupervised clustering, forecasting, and model evaluation.

## Contents
- supervised_learning.ipynb (classification & regression models) 
- unsupervised_learning.ipynb (clustering, anomaly detection, PCA)  
- forecasting_models.ipynb (time-series, ARIMA, Prophet, LSTM)  
- model_evaluation.ipynb (cross-validation, SHAP, feature importance)  

## Focus Areas
- Interpretability and feature importance  
- Production-quality evaluation standards  
- Realistic financial and operational datasets

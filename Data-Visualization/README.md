# Data Visualization

Dashboards and visualizations for exploratory analysis, KPI tracking, and presentation-ready analytics.

## Tools Demonstrated
- Tableau
- Python (Matplotlib, Seaborn, Plotly)
- Streamlit interactive dashboards
- Grafana for real-time monitoring

## Contents
- python_plots.ipynb (financial and statistical visualization examples)  
- streamlit_dashboard.py (multi-page interactive analytics app)  
- grafana_dashboard_config.json (sample operational dashboard configuration)
